## 🎯 使用方法

### 1. 编译 CSS

首先需要编译 Tailwind CSS：

```bash
npm run build
```

### 2. 查看页面

在浏览器中打开：
```
index-modern.html
```

### 3. 自定义内容

#### 修改颜色
在 `tailwind.config.js` 中修改：
```javascript
colors: {
  primary: "#3056D3",  // 主色
  secondary: "#13C296", // 次要色
}
```

#### 修改文字
直接在 `index-modern.html` 中修改对应的文字内容。

#### 添加图片
将团队成员照片放在 `assets/images/team/` 目录，然后更新 HTML 中的图片路径。

## 🐛 常见问题

**Q: 动画不工作？**
A: 确保已加载 WOW.js 和 animate.css

**Q: 样式不显示？**
A: 运行 `npm run build` 重新编译 CSS

**Q: 表单验证不工作？**
A: 确保已加载 form-validation.js

**Q: 图片不显示？**
A: 检查图片路径是否正确
