# TSu Energy 网站 - 快速开始指南

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 开发模式

```bash
npm run dev
```

这将启动 Tailwind CSS 的监听模式，自动编译样式文件。

### 3. 构建生产版本

```bash
npm run build:prod
```

这将生成优化后的生产版本文件。

## 🎨 新增的 JavaScript 模块

### 1. form-validation.js - 表单验证
自动验证表单输入，提供实时反馈。

**使用方法：**
```html
<script src="assets/js/form-validation.js"></script>
```

### 2. lazy-loading.js - 图片懒加载
优化图片加载，提升页面性能。

**使用方法：**
```html
<script src="assets/js/lazy-loading.js"></script>
<img data-src="path/to/image.jpg" alt="描述" />
```

### 3. language-switch.js - 语言切换
增强的多语言支持，带本地存储。

**使用方法：**
```html
<script src="assets/js/language-switch.js"></script>
<button data-language="en">English</button>
<button data-language="zh">中文</button>
```

### 4. component-loader.js - 组件加载器
动态加载 HTML 组件。

**使用方法：**
```html
<script src="assets/js/component-loader.js"></script>
<div id="navbar" data-component="templates/navbar.html"></div>
```

### 5. performance.js - 性能优化
提供性能优化工具函数。

**使用方法：**
```javascript
// Debounce 函数
const debouncedFn = PerformanceUtils.debounce(() => {
  console.log('执行！');
}, 300);

// Throttle 函数
const throttledFn = PerformanceUtils.throttle(() => {
  console.log('执行！');
}, 300);
```

### 6. accessibility.js - 无障碍功能
增强网站的无障碍访问性。

**使用方法：**
```html
<script src="assets/js/accessibility.js"></script>
```

## 🎯 如何在页面中使用新功能

### 在现有页面中添加新的 JS 文件

在 `<head>` 或 `</body>` 前添加：

```html

<script src="assets/js/performance.js"></script>

<!-- 核心功能 -->
<script src="assets/js/main.js"></script>

<!-- 表单验证 -->

<!-- 图片懒加载 -->

<!-- 语言切换 -->

<!-- 组件加载器 -->
<script src="assets/js/component-loader.js"></script>

<!-- 无障碍功能 -->
<script src="assets/js/accessibility.js"></script>
```

## 🌐 多语言支持

### 添加翻译

1. 在 HTML 元素上添加 `data-i18n` 属性：

```html
<h1 data-i18n="welcome">欢迎</h1>
```

2. 在语言 JSON 文件中添加对应的翻译：

```json
{
  "welcome": {
    "en": "Welcome",
    "zh": "欢迎"
  }
}
```

### 切换语言

```javascript
// 使用 JavaScript API
LanguageSwitcher.switchLanguage('zh');

// 或使用按钮
<button data-language="zh">中文</button>
```

## 🖼️ 图片优化

### 使用懒加载

将 `src` 改为 `data-src`：

```html
<!-- 之前 -->
<img src="image.jpg" alt="描述" />

<!-- 之后 -->
<img data-src="image.jpg" alt="描述" />
```

### 图片格式建议

- 照片：使用 WebP 或 JPEG
- 图标/Logo：使用 SVG
- 截图：使用 PNG

### 图片压缩

使用在线工具压缩图片：
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)

## 🐛 常见问题

### Q: Tailwind 类不生效？
A: 运行 `npm run build` 重新编译 CSS。

### Q: 组件没有加载？
A: 检查文件路径是否正确，确保 component-loader.js 已加载。

### Q: 表单验证不工作？
A: 确保 form-validation.js 已加载，输入框有 `required` 属性。

### Q: 图片没有懒加载？
A: 确保使用 `data-src` 而不是 `src`，并加载了 lazy-loading.js。
