## 🎯 如何使用新网站

### 方法 1: 直接使用新首页

1. 在浏览器中打开 `index-modern.html`
2. 查看效果
3. 如果满意，可以将其重命名为 `index.html` 替换原有首页

### 方法 2: 保留两个版本

1. 保持 `index.html` 为原有版本
2. 使用 `index-modern.html` 作为新版本
3. 在导航中添加切换链接

### 方法 3: 逐步迁移

1. 从 `index-modern.html` 中复制喜欢的部分
2. 粘贴到 `index.html` 中
3. 逐步替换各个 section

## 💡 自定义建议

### 颜色方案
在 `tailwind.config.js` 中修改：
```javascript
colors: {
  primary: "#3056D3",    // 改为你的品牌色
  secondary: "#13C296",  // 改为你的辅助色
}
```

### 字体
在 `src/tailwind.css` 中修改：
```css
@import url("https://fonts.googleapis.com/css2?family=YourFont");
```

### 动画速度
在 `tailwind.config.js` 中调整：
```javascript
animation: {
  'fade-in': 'fadeIn 0.5s ease-in-out',  // 调整时间
}
```
